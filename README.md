# Email Builder

Визуальный конструктор email-шаблонов на React: библиотека блоков, перетаскивание в «песочницу», настройка выбранного блока, предпросмотр и выгрузка готового HTML (с учётом тёмных шаблонов и preheader).

## Стек



- **React 19**, **Vite 7**
- **Sass** (модули + дизайн-токены в `src/styles/variables.scss`)
- **@dnd-kit/react** — drag-and-drop
- **Typograf** — типографика текста при сборке письма
- **ESLint 9**

## Требования

- Node.js **18+** (рекомендуется актуальный LTS)

## Установка и скрипты

```bash
npm install
npm run dev      # разработка, по умолчанию http://localhost:5173 (host: true в Vite)
npm run build    # production-сборка в dist/
npm run preview  # локальный просмотр собранного проекта
npm run lint     # ESLint
```

## Возможности

- **Меню** — добавление блоков из готовых паттернов в песочницу.
- **Песочница** — порядок блоков через DnD, выделение активного блока.
- **Панель инструментов** — форматирование текста, ссылки, дублирование и удаление блока (для активного шаблона).
- **Шапка**:
  - **Результат** — ввод preheader, сборка письма и скачивание `email.html`;
  - **Предпросмотр** — просмотр в модальном окне (десктоп / мобильная ширина);
  - **Восстановить** — загрузка песочницы из `localStorage` (кнопка неактивна, если сохранения нет);
  - **Сброс** — очистка песочницы и снятие сохранения.
- **Тёмная тема в HTML** — если среди блоков есть шаблоны в тёмном режиме, при экспорте в документ добавляются соответствующие стили (`hasDarkTemplates` / `getResult(..., { darkMode })`).

## Автосохранение (localStorage)

Состояние песочницы сериализуется в браузер при **ненулевом** списке шаблонов:

- Ключ: `email_builder:sandbox:v1` (см. `src/utils/constants.js`).
- Логика чтения/записи и статусов: `src/utils/sandboxStorage.js` (`EMPTY` / `OK` / `CORRUPTED`).

## Структура проекта (кратко)

```
src/
  App.jsx                 # корневой layout, провайдер DnD, автосохранение
  components/             # Header, Menu, Sandbox, Toolbar, Settings, Preview, Prompt, Toast
  patterns/               # данные и шаблоны блоков
  result/                 # сборка HTML: content.js, media.js, result.js
  utils/                  # константы, sandboxStorage, хелперы
  styles/                 # variables.scss и глобальные стили
```

## Конфигурация dev-сервера

В `vite.config.js` заданы `server.host: true` и `server.port: 5173` — удобно для доступа с других устройств в сети.
