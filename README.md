## Email Builder

Интерактивный конструктор HTML‑писем на React. Позволяет собирать письмо из готовых блоков (текст, заголовки, карточки, кнопки, шапки/футеры и готовые шаблоны), редактировать контент прямо в полотне и получать итоговый HTML‑код.

### Стек

- **React + Vite**
- **react-dnd** для drag‑and‑drop
- **SCSS‑модули** для стилизации

### Запуск проекта

```bash
npm install
npm run dev
```

После запуска откройте в браузере адрес, который покажет Vite (обычно `http://localhost:5173`).

### Продакшн‑сборка

```bash
npm run build
```

Собранные файлы попадут в папку `dist/`. Для предпросмотра продакшн‑сборки можно использовать любой статический сервер (например, `npx serve dist`).

### Основные возможности

- Перетаскивание блоков и шаблонов из левого меню в песочницу.
- Редактирование текста прямо в блоках (contenteditable).
- Панель форматирования текста (жирный, курсив, ссылки).
- Настройки выбранного блока (отступы, шрифты, цвета, ссылки и т.п.).
- Сохранение состояния и темы в `localStorage` и восстановление по кнопке.
- Экспорт готового письма в HTML‑файл.

### Структура проекта (основное)

- `src/App.jsx` — корневой компонент приложения.
- `src/components/` — UI‑компоненты (шапка, меню, песочница, тулбар, настройки и т.д.).
- `src/templates/` — описание блоков и готовых шаблонов писем.
- `src/utils/` — утилиты (работа с `localStorage`, DOM‑хелперы, общие функции).

### Замечания по разработке

- Старайтесь добавлять новые блоки через `src/templates/blocks/` и регистрировать их в `blocksRegistry`.
- Для тулбара используйте `toolbarPreset` в описании блока, чтобы управлять набором кнопок.
- При изменении формата сохраняемых данных обновляйте утилиту `src/utils/storage.js`, сохраняя обратную совместимость по возможности.
